Fix It : A web application that solves little problems such as :
  -painting the walls,
  -Fixing electricity ,
  -Gardening....

  Each visitor is capable of creating his own profile wich contains :
  -personal Information(firstname,lastname,email,phone,...)
  -The things that he can offer or lend.
  -The aid that he can give.

  After getting in the main page of the site :
  the visitor is asked to search for his own needs.
  the result contains the profile of the poster and his location that will be shown in a map.
